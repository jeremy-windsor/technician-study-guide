# TTS Pronunciation Guide

Rules for text-to-speech generation of the Technician study guide.

## Approach

**Expand** acronyms that teach meaning (students need to know what it stands for).
**Spell out** acronyms that are common labels (everyone knows them by letters).
**Say naturally** acronyms that have standard spoken pronunciations.

## Special Cases

| Written | Say |
|---------|-----|
| ✅ | (silence — skip this symbol) |
| > | (silence — start of explanation block) |
| ### | (silence — question header) |
| ## | (pause — section break) |

| **bold text** | (read normally, no emphasis change needed) |

| 47 CFR | forty seven C F R |
| FCC Rule | F C C rule |

| W1XX | W one X X |
