# TTS Pronunciation Dictionary

Pronunciation guide for text-to-speech engines reading this study guide. Format: `TERM → spoken form`

|------|---------------|
| Hz | hertz |
| kHz | kilohertz |
| MHz | megahertz |
| GHz | gigahertz |
| HF | H-F (spell out) |
| VHF | V-H-F |
| UHF | U-H-F |
| SHF | S-H-F |
| EHF | E-H-F |
| RF | R-F |
| dB | decibels |
| dBm | D-B-M |
| dBi | D-B-I |
| CW | C-W |
| FM | F-M |
| AM | A-M |
| SSB | single sideband |
| USB | upper sideband |
| LSB | lower sideband |
| PEP | peak envelope power |
| SWR | S-W-R |
| VSWR | V-S-W-R |
| VFO | V-F-O |
| AGC | A-G-C |
| RIT | R-I-T |

## Formulas (read as)

| Written | Spoken |
|---------|--------|
| E = I × R | E equals I times R |
| I = E / R | I equals E divided by R |
| P = E × I | P equals E times I |
| P = I²R | P equals I squared times R |
| P = E²/R | P equals E squared divided by R |
| λ = 300/f | lambda equals 300 divided by frequency in megahertz |
